# SubnetStreak
> Tôi vibe code á :33 (Tôi đang ôn bài thì nãy ra ý tưởng này thôi)

>Nền tảng luyện IPv4 Subnetting cho CCNA với câu hỏi ngẫu nhiên, lời giải từng bước, XP, streak và theo dõi tiến độ ngay trên trình duyệt.

## Tính năng

- Luyện subnetting không giới hạn: network, broadcast, CIDR, wildcard, VLSM và longest prefix match.
- Chế độ Daily Challenge, Speed Run, Survival, Time Attack, Exam và Boss Challenge.
- Lưu lịch sử, điểm XP, streak, cài đặt và xuất/nhập dữ liệu bằng LocalStorage.
- Bộ kiểm thử xác thực 10.000 phép tính subnet và 10.000 câu hỏi sinh tự động.
